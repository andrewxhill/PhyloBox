Embedding phylogenies in on your webpage is what PhyloBox is all about. If the methods we show you here don't work for your particular project, please contact me, AndrewXHill@gmail.com and I will do my best to help you make it happen.

Embedding a tree in a site you fully control different that embedding a tree in a more restrictive site, say Blogger. For instruction on getting your tree into Blogger, go here.

The easiest way to embed your tree in HTML, is to simple cut and paste the code from the 'Embed Tree' link within the Editor directly into your web page source code. See example code to right.

You can change the height and width parameters to your liking, and by wrapping it in a custom DIV element you can easily position it anywhere on the page. 

<div width="400" 
     height="385"
     style="width:400px;height:385px;"
     id="phylobox_perm783d06b1-e040-4c94-bf18-8a35bda8ea00"
     class="phylobox_embed_parent"> <a href="http://phylobox.appspot.com"> <img src="http://phylobox.appspot.com/static/images/widget/holder_425.png" width="375" height"344" /> </a> </div>
